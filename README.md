# GSMU Skills

Repositorio privado de **skills técnicas y reglas de proyecto** usadas por
agentes de IA y desarrolladores humanos dentro del ecosistema GSM Universe.

Este repo define:
- stack oficial
- librerías permitidas
- reglas de arquitectura
- estilos de código
- patrones reutilizables

## Uso recomendado

- Mantener este repo como **fuente única de verdad**
- Integrarlo en proyectos mediante:
  - git submodule (recomendado)
  - copia local (snapshot)
- Cada proyecto debe tener su propio `project_context.md`

## Filosofía
Consistencia > innovación  
Robustez > moda  
Simpleza > sobreingeniería

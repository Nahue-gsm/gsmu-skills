# Skill: Backend PHP Baseline (GSMU)

## Objetivo
Definir endpoints simples y seguros para formularios/CRUD básicos.

## Reglas
- Validación server-side obligatoria
- Prepared statements (MySQLi/PDO) siempre
- Logs sin secretos
- Respuestas JSON consistentes (status + message + data)

## Estructura recomendada
- `/api/` endpoints
- `/config/` conexiones
- `/services/` lógica

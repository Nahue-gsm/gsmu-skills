# Skill Global: Reglas del Proyecto (GSM Universe Stack)

Objetivo:
Unificar criterios técnicos para todos los proyectos y agentes.

Principios:
- simple > complejo
- robusto > moderno
- mantenible > elegante
- pocas dependencias > muchos frameworks

Stack oficial:
- Electron (desktop)
- PHP nativo (backend)
- MySQL / MariaDB
- JSON como storage liviano
- PowerShell / CMD / WinPE
- excel4node
- electron-builder

Prohibido:
- Laravel / Symfony
- React / Vue / Angular
- Electron Forge
- ORMs complejos
- Docker (salvo pedido explícito)

Regla:
Si no está definido acá o en el project_context.md, NO se usa.

# Skill: Landing Page Baseline (GSMU)

## Objetivo
Crear landings rápidas, limpias y mantenibles, respetando limitaciones de hosting.

## Preguntas mínimas
- ¿Hosting soporta: PHP / Node / solo estático?
- ¿Formularios: email, whatsapp, DB?
- ¿SEO es prioridad?
- ¿Analítica: GA/Pixel?

## Reglas
- Performance primero (imágenes optimizadas, lazy loading)
- Evitar dependencias pesadas si no aportan valor
- Formularios siempre con validación server-side si hay backend
